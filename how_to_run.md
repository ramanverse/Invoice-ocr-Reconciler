Bhai, is website ko chalane ke liye aapko do cheezein karni hongi: Backend start karna aur Frontend start karna.

### Prerequisites:
Aapke computer mein **Node.js** install hona chahiye.

---

### Step 1: Backend Start Karo ğŸ–¥ï¸
Sabse pehle server shuru karna hoga taki data load ho sake.
1. Ek naya terminal open karo.
2. `backend` folder ke andar jao:
   ```bash
   cd backend
   ```
3. Agar pehli baar chala rahe ho, to dependencies install karo:
   ```bash
   npm install
   ```
4. Server start karo:
   ```bash
   npm start
   ```
   *(Aapka server `http://localhost:5001` par chalne lagega)*

---

### Step 2: Frontend Start Karo ğŸŒ
Ab UI (website) shuru karte hain.
1. Ek aur naya terminal open karo.
2. `frontend` folder ke andar jao:
   ```bash
   cd frontend
   ```
3. Agar pehli baar chala rahe ho, to dependencies install karo:
   ```bash
   npm install
   ```
4. Website start karo:
   ```bash
   npm run dev
   ```
   *(Terminal mein ek link aayegi, usually `http://localhost:5173`. Us par click karo!)*
