To run this application, you need to start both the Backend and the Frontend servers.

### Prerequisites:
Ensure that **Node.js** is installed on your computer.

---

### Step 1: Start the Backend ğŸ–¥ï¸
First, start the backend server to enable data processing and storage.
1. Open a new terminal window.
2. Navigate to the `backend` folder:
   ```bash
   cd backend
   ```
3. If this is your first time running the app, install the dependencies:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   npm start
   ```
   *(Your server will be running at `http://localhost:5001`)*

---

### Step 2: Start the Frontend ğŸŒ
Now, start the user interface.
1. Open a second terminal window.
2. Navigate to the `frontend` folder:
   ```bash
   cd frontend
   ```
3. If this is your first time running the app, install the dependencies:
   ```bash
   npm install
   ```
4. Start the website:
   ```bash
   npm run dev
   ```
   *(Click the link provided in the terminal, usually `http://localhost:5173`)*
